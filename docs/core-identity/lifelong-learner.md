## 核心定位

终身学习者是uWise 百科的核心驱动力。在快速变化的时代，持续学习不仅是职业需求，更是生活态度。通过跨领域学习，不断拓展认知边界。

## 学习理念

> 学而不思则罔，思而不学则殆。——《论语》

### 核心原则

1. **学以致用** - 将知识转化为实践能力
2. **跨界融合** - 打破学科边界，融会贯通
3. **反思迭代** - 持续反思，不断优化学习方法
4. **输出倒逼** - 通过教授他人来深化理解
