# 🏗️ ∑ 架构师

> **做正确的事情** - 负责设计和规划复杂系统的架构

## 核心定位

架构师不仅是技术专家，更是系统思维的实践者。通过25年的职业迭代，从飞机工程到AI专家，形成了独特的跨领域架构思维。

## 职业发展路径

```
飞机工程(1999) → 软件交付(2007) → 运维管理(2011) → 项目管理(2016) → 系统架构师(2020) → AI专家(2024)
```

### 关键转型节点

| 时期 | 转型 | 核心收获 |
|------|------|---------|
| 1999-2006 | 航空电气→计算机服务 | 系统性思维、严谨工程方法 |
| 2007-2010 | 软件交付 | 敏捷开发、服务交付 |
| 2011-2015 | 运维管理 | 监控体系、稳定性保障 |
| 2016-2019 | 项目管理 | 资源协调、风险管理 |
| 2020-2023 | 系统架构师 | 架构设计、技术决策 |
| 2024-今 | AI专家 | 大模型应用、AI原生 |
