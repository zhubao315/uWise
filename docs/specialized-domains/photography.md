# 📷 摄影家

> **用光影讲述故事** - 捕捉瞬间，创造永恒

## 领域概述

摄影是技术与艺术的完美结合，作为生活艺术家的重要组成，通过镜头记录生活之美，表达个人视角。

## 摄影基础

### 构图法则
- **三分法**：黄金分割点构图
- **引导线**：利用线条引导视线
- **框架构图**：利用前景创造层次
- **对称构图**：平衡与和谐

### 光线运用
- 黄金时段（日出日落）
- 蓝调时刻
- 逆光剪影
- 人工光源

### 曝光三要素
```
光圈 + 快门 + ISO = 正确曝光
```

## 摄影类型

### 风光摄影
- 自然景观
- 城市建筑
- 星空摄影

### 人文摄影
- 街头摄影
- 人物肖像
- 纪实摄影

### 技术摄影
- 微距摄影
- 长曝光
- HDR合成

## 后期处理

### 工具
- **Lightroom**：基础调整
- **Photoshop**：精细修图
- **Capture One**：专业调色

### 工作流
```
导入 → 筛选 → 调色 → 精修 → 导出
```
