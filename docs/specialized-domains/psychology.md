# 🧠 心理学专家

> **理解人心的科学** - 认知、情绪与行为的探索

## 领域概述

心理学是理解人类思维、情感和行为的科学，作为终身学习者的重要领域，帮助更好地理解自己和他人。

## 核心领域

### 认知心理学
- **注意力**：选择性注意、分配性注意
- **记忆**：工作记忆、长期记忆
- **决策**：启发式、认知偏差
- **学习**：条件反射、观察学习

### 社会心理学
- 从众与服从
- 说服与影响
- 群体行为
- 偏见与歧视

### 发展心理学
- 认知发展阶段
- 人格形成
- 终身发展

## 应用领域

### 情绪管理
```
认知重评 → 情绪调节 → 行为改变
```

### 决策优化
- 识别认知偏差
- 理性决策框架
- 风险感知

### 人际关系
- 沟通技巧
- 冲突解决
- 团队协作

## 经典理论

### 马斯洛需求层次
```
自我实现 → 尊重需求 → 归属需求 → 安全需求 → 生理需求
```

### 双系统思维（卡尼曼）
- **系统1**：快速、直觉、自动
- **系统2**：缓慢、理性、控制
