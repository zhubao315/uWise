# ☁️ 云计算与云原生

> **数字化转型的基石** - 从单体应用到云原生架构

## 领域概述

云计算与云原生是现代架构师的核心技能，涵盖容器化、微服务、DevOps等关键技术。

## 技术栈

### 容器化技术
- **Docker**：容器化基础
- **Kubernetes**：容器编排
- **Helm**：包管理
- **Istio**：服务网格

### 微服务架构
- 服务拆分原则
- API网关设计
- 服务发现与注册
- 分布式事务

### DevOps实践
- CI/CD流水线
- GitOps
- 监控与可观测性
- 混沌工程

### Serverless
- 函数计算
- 事件驱动架构
- 无服务器数据库
