# 📊 金融市场

> **资本市场的智慧** - 理解市场规律，把握投资机会

## 领域概述

金融市场是投资人的核心领域，涵盖股票、债券、衍生品等多种资产类别的分析与投资。

## 市场结构

### 股票市场
- **A股市场**：沪深两市、创业板、科创板
- **港股市场**：恒生指数、科技股
- **美股市场**：纳斯达克、纽交所

### 债券市场
- 国债、企业债、可转债
- 利率分析
- 信用评级

### 衍生品市场
- 期货、期权
- 对冲策略
- 风险管理
