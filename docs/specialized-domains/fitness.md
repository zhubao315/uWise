# 🏃 健身跑者

> **用脚步丈量世界** - 奔跑中的生命力量

## 领域概述

跑步是最简单也最有效的运动方式，作为生活艺术家的重要组成，通过系统训练和比赛，挑战身体极限，获得身心的全面提升。

## 跑步基础

### 跑姿要领
- 身体微微前倾
- 落地轻盈
- 步频180步/分钟
- 手臂自然摆动

### 训练方法
- **轻松跑**：有氧基础
- **节奏跑**：乳酸阈值
- **间歇跑**：速度提升
- **长距离**：耐力建设

### 马拉松训练
```
基础期(4周) → 提升期(4周) → 巅峰期(4周) → 减量期(4周)
```
