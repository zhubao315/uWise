# 📱 产品经理

> **连接用户与技术的桥梁** - 以用户为中心的产品设计

## 领域概述

产品经理是架构师的重要子领域，负责定义产品方向、设计用户体验、协调团队开发，将用户需求转化为技术实现。

## 核心能力

### 用户研究
- 用户访谈
- 用户画像
- 需求分析
- 竞品分析

### 产品设计
- 产品定位
- 功能规划
- 交互设计
- 原型制作

### 数据驱动
- 数据埋点
- 数据分析
- A/B测试
- 转化优化

## 产品方法论

### 设计思维
```
同理心 → 定义问题 → 创意构思 → 原型制作 → 测试验证
```

### 精益创业
```
构建 → 测量 → 学习 → 迭代
```

### 用户故事
```
作为<用户角色>，我想要<功能>，以便<价值>
```
