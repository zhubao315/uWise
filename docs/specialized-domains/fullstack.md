# 🔧 全栈工程师

> **前后端通吃的实战派** - 从界面到数据库的完整掌控

## 领域概述

全栈工程师是架构师的重要子领域，具备前端、后端、数据库等全方位的开发能力，能够独立完成完整的应用开发。

### 前端技术
- **框架**：React、Vue、Angular
- **状态管理**：Redux、Vuex、Pinia
- **构建工具**：Vite、Webpack
- **CSS框架**：Tailwind CSS、Ant Design

### 后端技术
- **语言**：Python、Go、Java、Node.js
- **框架**：FastAPI、Gin、Spring Boot
- **API设计**：RESTful、GraphQL
- **消息队列**：Kafka、RabbitMQ

### 数据库
- **关系型**：PostgreSQL、MySQL
- **NoSQL**：MongoDB、Redis
- **向量数据库**：Milvus、Pinecone

### DevOps
- **容器化**：Docker、Kubernetes
- **CI/CD**：GitHub Actions、GitLab CI
- **监控**：Prometheus、Grafana
