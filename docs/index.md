# uWisdom 百科

> **AI驱动的领域知识百科** | 持续进化，与时俱进

## 🎯 核心理念

**uWisdom百科**是一个基于四大核心身份的持续进化知识体系，采用领域驱动框架（Area → Project → Task），将知识转化为实际行动。

> 成果源于**挖掘机会**，而非解决问题。——德鲁克《成果管理》(1964)

## 🔄 领域驱动框架

```
Area (领域) → Project (项目) → Task (任务)
```

### 示例

| 身份 | Area（领域） | Project（项目） | Task（任务） |
|------|-------------|----------------|-------------|
| 架构师 | 系统架构设计 | AI原生转型 | 评估大模型选型、设计Prompt工程框架 |
| 投资人 | 投资组合管理 | 量化策略开发 | 研究因子模型、回测交易策略 |
| 终身学习者 | 持续技能提升 | 国学研究 | 研读《论语》、实践传统智慧 |
| 生活艺术家 | 生活品质提升 | 葡萄酒探索 | 学习品鉴技巧、探索世界产区 |
