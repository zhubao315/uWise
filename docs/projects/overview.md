# 📊 项目实践总览

> **从知识到行动** - Area → Project → Task 的实践框架

## 核心理念

uWisdom百科不仅是知识的积累，更是实践的指南。每个领域都通过项目化的方式，将知识转化为实际行动。

## 领域驱动框架

```
Area (领域) → Project (项目) → Task (任务)
```

### 框架说明

| 层级 | 定义 | 示例 |
|------|------|------|
| Area | 大的领域方向 | 系统架构、投资管理、生活品质 |
| Project | 具体项目 | AI平台开发、量化策略、马拉松训练 |
| Task | 可执行任务 | 设计RAG架构、回测交易策略、完成16周训练 |
